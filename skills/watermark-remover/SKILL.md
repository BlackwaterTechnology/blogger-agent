# Watermark Remover

## Overview
Specifically designed to remove the "AI生成" (AI-generated) watermark from Google NotebookLM cinematic videos.

## Core Workflow

### NotebookLM Video Cleaning (Mandatory)
NotebookLM videos (1280x720) consistently place the "AI生成" watermark in the bottom-right corner. Use the following fixed-region command:

```bash
python3 watermark_remover.py input.mp4 --model telea -r "1100,660,160,50" -o output_clean.mp4
```

## Policy
- **NotebookLM Only:** This skill should only be applied to videos generated via the `generate-video` skill (NotebookLM).
- **No Auto-detection:** Do not rely on automatic watermark detection as it is prone to false negatives for this specific watermark. Always use the `-r "1100,660,160,50"` manual region.

## Parameters

| Parameter | Value | Purpose |
|---|---|---|
| `-r, --region` | `1100,660,160,50` | Fixed coordinates for NotebookLM 720p watermark. |
| `--model` | `telea` | Fast, CPU-based inpainting. |
| `-o, --output` | `path/to/video_clean.mp4` | Clean output path. |
